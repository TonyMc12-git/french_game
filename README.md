# French Game

A touch-first beginner French quiz built around short A-level words and phrases.

## How It Works

Each round shows one French word or phrase.

You choose every English translation that fits from 20 options, then tap `Check answer`.

## Local Testing

Run a local static server from this folder:

```powershell
python -m http.server 8012
```

Then open:

```text
http://localhost:8012/index.html
```

## Phone Install

After deployment to GitHub Pages, open:

- `https://tonymc12-git.github.io/french_game/`

Then use `Add to Home Screen` on your phone.
